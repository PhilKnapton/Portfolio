### Pulmonary Fibrosis Resource Website

### Overview
This project involved designing and developing a pulmonary fibrosis resource website using the Wix platform. The goal was to create an accessible online space where users can find educational resources, participate in discussions, and stay informed about community events related to pulmonary fibrosis. The platform includes multi-user account functionality, community discussion features, blog-style posts, and event pages with reminder capabilities to help foster engagement and support among members of the pulmonary fibrosis community. The website is currently in an active development and testing phase as additional features and content continue to be refined.

https://vipfsupport.wixsite.com/vi-pf-support

### Features
- Multi-user account system with login and profile access  
- Events page for sharing and viewing upcoming community events  
- Reminder functionality to help users track important events  
- Comment sections allowing users to interact and share experiences  
- Informational pages providing resources and awareness content related to pulmonary fibrosis  

### Purpose
The website was designed to provide both educational information and community engagement tools. By combining informational content with interactive features such as events, reminders, and comment discussions, the platform encourages users to stay informed and connected with others affected by pulmonary fibrosis.

### Technologies Used
- Wix Website Builder
- Wix user account management
- Wix event and community features

### Skills Demonstrated
- Website design and layout development
- Multi-user website configuration
- Community-focused web functionality
- Content organization and user engagement features
